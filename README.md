# FaceApp Baidu

FaceApp integrates **Face Recognition, Fingerprint, and RFID**
with access-level control on the **RV1109** platform.

## Features
- Face recognition (Baidu SDK)
- Fingerprint authentication
- RFID access
- Access-level based permissions
- New unified UI
- RV1109 embedded Linux support

## Platform
- Rockchip RV1109
- Embedded Linux
- Qt-based UI
